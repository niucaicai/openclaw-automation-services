## 2. 费用与支付

### 2.1 费用标准
- 一次性项目：人民币 ______ 元
- 月费订阅：人民币 ______ 元/月（首月预付，后续按月续费）

### 2.2 支付方式
- 甲方通过支付宝/微信/银行转账支付至乙方指定中转账户

## 3. 交付与验收

### 3.1 交付物
- 完整源代码（Python脚本/项目）
- 使用文档（Markdown/PDF）
- 视频教程（如有）

### 3.2 交付时间
- 开发周期：______ 天（自乙方收到款项起计算）

- 脚本能正常运行，采集数据符合需求
- 甲方可提出7天内免费技术支持（bug修复、小调整）
