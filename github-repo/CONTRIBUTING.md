# 如何开始合作

感谢您对 OpenClaw 自动化服务的兴趣！以下是合作流程：

---

## 1. 发送需求

请将您的需求描述发送到：**automation.openclaw@gmail.com**

建议包含以下信息（[复制模板](#需求模板)）：

```
项目名称：
目标网站：
需要采集的数据：
采集频率：
数据格式：
特殊需求：（如需要登录、处理验证码等）
预算范围：
期望交付时间：
```

---

## 2. 等待报价

- 我们会在 **1小时内** 回复详细方案和报价
- 如需进一步沟通，我们会预约时间（电话/微信）

---

## 3. 确认并付款

- 您确认报价后，付款到指定中转账户
- 付款后我们收到 **80%**，渠道方留存 **20%**
- 付款完成后，我们开始开发

---

## 4. 交付与试用

- 开发周期：2-7天（视复杂度）
- 交付内容：源代码 + 使用文档 + 视频教程
- 7天免费技术支持（bug修复、小调整）

---

## 5. 长期合作

- 一次性项目：交付后合作结束（可选择维护服务）
- 月费订阅：首月预付，次月起每月1号续费
- 老客户享受优惠：转介绍返现30%、季度9折、年度8折

---

## 需求模板

```
项目名称：电商价格监控

目标网站：
- https://item.taobao.com/item.htm?id=123456
- https://item.taobao.com/item.htm?id=789012

需要采集的数据：
- 商品标题
- 当前价格
- 累计销量
- 评价数量
- 店铺名称

采集频率：每天1次（早上10点）

数据格式：Excel文件，包含所有字段和时间戳

特殊需求：
- 价格变动超过5%时，微信通知
- 需要处理淘宝的反爬（可能需要Selenium）

预算范围：1500元/月（订阅）或2000元（一次性）

期望交付时间：3天内
```
